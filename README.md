# 🧮 Starknet Counter DApp

This is a simple **Counter DApp** built using **Cairo Smart Contracts** and a **React.js frontend**, designed for the **Starknet** ecosystem.

---

## 🚀 Features

- Connect your **Argent X** or **Braavos** wallet
- Interact with a deployed **Cairo smart contract** on Starknet
- Set a counter onchain
- View the current counter state

---

## 📦 Getting Started

### 📁 Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/Starknet-Dapp.git
cd Starknet-Dapp
```

### 📦 Step 2: Install Dependencies

Make sure you're in the project directory:

```bash
npm install
```

### 🔧 Step 3: Run the Development Server

```bash
npm run dev
```

The app will be running at `http://localhost:5173`

---

## 🧙 Prerequisites

Before using the DApp, ensure you have one of the following **Starknet-compatible wallets** installed in your browser:

- 🔐 [Argent X](https://www.argent.xyz/argent-x/)
- 🦊 [Braavos](https://braavos.app/)

Connect your wallet before interacting with the contract.

---

## 🛠️ Technologies Used

- **[Cairo](https://www.cairo-lang.org/)** – Smart contract language for Starknet
- **[React.js](https://reactjs.org/)** – Frontend UI framework
- **[Starknet.js](https://www.starknetjs.com/)** – Starknet JavaScript SDK
- **[Vite](https://vitejs.dev/)** – Lightning-fast build tool
